# liruijian.com Personal Site V1

这是 `liruijian.com` 的个人网站静态版本。当前仓库已经从“单个项目站”切换为“个人总站 + 项目归档”的结构。

## 当前结构

- `index.html`：个人站首页 V1
- `styles.css`：个人站首页样式
- `app.js`：导航交互与滚动动效
- `hackathon.html`：原黑客松项目详情页归档入口
- `hackathon.css`：黑客松项目页样式
- `hackathon.js`：黑客松项目页交互
- `DESIGN.md`：站点定位、信息架构与视觉方向说明

## 这次改动的核心

- 根域首页不再只服务某一个项目
- 首页改为承载个人定位、精选作品、可体验应用、Notes 和关于
- 原有 `AI+体育 Vibe Coding 训练营暨创新黑客松` 页面被保留为独立项目详情页
- 站点结构开始朝长期个人品牌站演进，而不是一次性活动页

## 本地预览

在当前目录执行：

```bash
python3 -m http.server 4323
```

然后访问：

```text
http://127.0.0.1:4323
```

黑客松详情页地址：

```text
http://127.0.0.1:4323/hackathon.html
```
