# DESIGN

## 项目定位

这是一个单页招生简章网站，目标不是做学院门户，而是为一个具体研讨班服务：

- 主题：`AI+体育应用开发实战研讨班`
- 目标用户：全国体育院校师生，以及有明确体育场景需求的行业实践者
- 页面任务：在最短时间内说明项目价值、两天安排、20个应用场景、收费政策与报名要点

## 视觉方向

页面采用“高校权威感 + 体育现场感 + AI工具感”的混合方向：

- 主色：深海军蓝、竞技酒红、金色
- 辅色：草地绿少量点缀，避免页面过于学院化或过于科技模板化
- 背景：暖纸色正文区 + 深色英雄区，形成招生页的主次层次
- 质感：用玻璃面板、轨道线条、光晕与印刷纸感结合，做出“训练场 + 招生简章”的混合气质

## 字体策略

- 标题：中文衬线字体栈，增强正式感与机构气质
- 正文：中文无衬线字体栈，保证长文本可读性
- 避免纯系统默认网页观感

## 版式结构

单页从“先决策，再细看”的顺序展开：

1. Hero：一句话说明项目是什么、值不值得看
2. 项目概览：解释课程为什么不同
3. 两天安排：建立“这是实战课”的信任感
4. 20个应用：证明内容密度和落地面
5. 合作建议：提升项目外延与机构合作想象空间
6. 教练位：预留后续正式发布需要的师资信息
7. 收费与报名：集中放政策、优惠和占位信息
8. FAQ：提前消化关键异议

## 内容策略

文案不走“宣传口号堆叠”路线，而强调三点：

- 学员能做出什么
- 课程到底怎么教
- 这件事为什么和体育人直接相关

同时把用户没有提供的信息显式标注为“待补充”，避免误以为是最终版正式简章。

## 动效策略

- 只保留少量有意义的入场动画
- 导航随滚动高亮，帮助长页浏览
- 移动端导航使用简洁折叠菜单

## 后续建议

如果继续迭代下一版，优先补以下内容：

- 正式开班日期与地点
- 报名二维码 / 表单链接
- 三位教练正式信息与头像
- 主办单位落款、联系人与咨询电话
- 如需正式发布，可再增加“往届案例”或“应用演示截图”增强信任
